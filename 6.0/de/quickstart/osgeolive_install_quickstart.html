OSGeo-Live auf einer Festplatte installieren¶

OSGeo-Live kann auf die gleiche Art auf Ihrer Festplatte installiert werden, wie auch die Ubuntu Distribution installiert wird.

Dies kann auch innerhalb einer virtuellen Maschine (VM) passieren, um eine permanente OSGeo-Live VM aus einem DVD oder ISO Abbild zu erzeugen.

Systemvoraussetzungen¶

  • 10 GB freier Festplattenspeicher

Install OSGeo-Live¶

Wählen Sie Install RELEASE

Geben Sie als Password “user” (ohne Anführungszeichen) ein

Wählen Sie die gewünschte Sprache

Wählen Sie die Zeitzone

Wählen Sie die Tastatur

Geben Sie an, auf welche Festplatte Sie installieren möchten (Achtung, überschreiben Sie nicht aus Versehen eine existierende Partition).

Standardeinstellungen sind:

  • Name = “user”
  • username = “user”
  • password = “user”
  • computer name = “osgeolive”
  • Log In automatically

Wählen Sie Install

Holen Sie sich einen Kaffee, während die Installation durchgeführt wird.

Das war es schon.
