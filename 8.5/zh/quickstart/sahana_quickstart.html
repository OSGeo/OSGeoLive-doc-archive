Sahana 快速入门¶

演示¶

　　Sahana Eden 在 OSGeo Live 已经安装并配置。请通过 Geospatial 菜单的 Crisis Management 目录或 这个 URL 访问。在登录前其信息是只读的。对于新建和编辑记录，需要以以下用户登录：

  • 用户名/Username: admin
  • 密码/Password: admin

　　若在 Ushahidi 快速入门中已经输入了示例数据，则通过 该链接 可以访问。

　　Sahana 已为 OSGeo Live 内置的数据集配置，并可以使用 OpenStreetMap 作为 地图浏览客户端 的底图。Sahana Eden 支持多种 Open Street Map 服务。

项目开发¶

　　Sahana Eden 一直在寻找更多的开发人员参与该项目。其主要开发工具为 Python 语言和 Web2Py 框架。项目同样希望熟悉测绘工程、JQuery 和 JavaScript 的人员参与。

  • Sahana Eden 社区参与页面
  • Ticket 系统 用于报告问题
