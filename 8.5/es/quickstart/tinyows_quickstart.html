Guía de inicio rápido de TinyOWS¶

TinyOWS es un Servicio Web de Eventos Transaccional (WFS-T) de alto rendimiento, ligero y fácil de desplegar mediante una interfaz CGI o FastCGI y usando PostGIS para el almacenamiento de datos.

Esta guía describe como:

  • mostrar una capa WFS en QGIS
  • editar una capa WFS-T en QGIS

Mostrar una capa WFS en QGIS¶

  1. Abrir el programa QGIS Geospatial ‣ Desktop GIS ‣ Quantum GIS.
  2. Pulsar Capa ‣ Añadir capa PostGIS....
    • Primero mostramos la capa original en PostGIS.
  3. Pulsar Nuevo.
  4. Escribir pgrouting como nombre de la Conexión y Base de Datos.
  5. Pulsar Aceptar.
  6. Pulsar Conectar.
  7. Seleccionar ways en el listado de tablas.
  8. Presionar Añadir.
  9. Hacer zoom
  10. Seleccionar Complementos -> Administrar complementos -> Complemento de WFS -> Aceptar
    • Habilitar el “WFS Layer plugin”.
  11. Hacer clic en Capa ‣ Añadir capa WFS....
    • Ahora se añade la capa WFS basada en la misma tabla.
  12. Pulsar Nuevo.
  13. Escribir tinyows como nombre y http://localhost/cgi-bin/tinyows como URL
  14. Pulsar Aceptar.
  15. Pulsar Conectar.
  16. Seleccionar ows:ways en la lista de capas.
  17. Habilitar la opción Solicitar sólo objetos espaciales que solapen con la extensión de la vista actual.
  18. Pulsar Aceptar.
    • Ahora la capa WFS layer es mostrada:

Editar datos con WFS-T en QGIS¶

  1. Hacer clic Capa ‣ Conmutar Edición.
  2. Hacer clic Edición ‣ Herramienta de Nodos.
  3. Mover un vértice (aspa) a otra posición.
  4. Hacer clic Capa ‣ Guardar Edición.
  5. Oculta la capa desmarcando tows:ways en el listado de capas.
    • El elemento editado se debería ver con un color diferente procedente directamente de la base de datos PostGIS.

Más información¶

Para aprender más sobre TinyOWS, un punto de partida es la TinyOWS Documentation en la página web de MapServer.
