QGIS Server¶

Web Service¶

QGIS Server fournit un service web cartographique (WMS) utilisant les mêmes libraries que l’application bureautique Quantum GIS (QGIS). Les cartes et les modèles d’impression créés dans QGIS peuvent être publiés comme des cartes web simplement en copiant le fichier de projet de QGIS dans le répertoire du serveur. Les cartes web résultantes ressemblent exactement à celles de l’outil bureautique. QGIS Server fonctionne habituellement comme un module CGI/FastCGI dans le serveur web Apache.

Caractéristiques principales¶

  • implémentation du Web Map Service (WMS)
  • sortie au format PDF
  • création de carte WYSIWYG avec QGIS
  • symbolisation cartographique avancée
  • support du Styled Layer Descriptor (SLD)

Standards implémentés¶

  • OGC Web Map Service (WMS) 1.3.0
  • OGC Styled Layer Descriptor (SLD) 1.0
