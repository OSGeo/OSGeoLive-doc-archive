OSGeo-Live Internationalisation Quickstart¶

Cambiare Lingua¶

Molte lingue comuni sono incluse nella OSGeo-Live. Per cambiare lingua:

  1. Log out
  2. Selezionare altra lingua dal menu in basso nello schermo.
  3. Selezionare user=”user”, password=”user”
  4. Ripetere il processo se volete cambiare lingua di nuovo.

I menu di sistema e molte delle applicazioni appariranno nella lingua scelta. Se uno specifico programma continua ad essere in Inglese, vuol dire che il programma non possiede quella traduzione disponibile - per favore considerate di contribuire volantariamente alla traduzione. Contattare il progetto considerato direttamente per maggiori informazioni.

Per maggiori lingue vedere Applications ‣ System ‣ Language Support (Richiede una connessione ad internet).

Supporto alla tastiera¶

Il supporto alle tastiere internazionali è installato, ma si dovrà configurare il sistema per la vostra specifica tastiera. Per cambiare le impostazioni aprire Applications ‣ Settings ‣ Settings Manager quindi fate doppio click su Tastiera nella finestra seguente. La vostra password sarà richiesta per permettere l’aggiornamento della configurazione, digitare “user” per l’attuale utente “user”.

  1. Scegliere la scheda Layout nella finestra Tastiera
  2. Disabilitare l’opzione Use system defaults
  3. Cliccare sul pulsante Add per aggiungere un ulteriore layout
  4. Scegliere il layout preferito e
  5. Selezionarlo nella finestra Tastiera
  6. Tornare indietro alla pagina principale delle impostazioni (Overview)
  7. Close la finestra delle Impostazioni e log out per applicare le modifiche
