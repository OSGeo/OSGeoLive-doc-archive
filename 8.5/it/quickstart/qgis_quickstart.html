Guida rapida su QGIS¶

Quantum GIS (QGIS) è un client GIS Desktop user friendly ed open-source per la gestione, visualizzazione, modifica, analisi di dati geografici e per la composizione di mappe per la stampa.

Questa breve introduzione all’uso descrive come:

  • aggiornare un progetto QGIS esistente
  • creare un nuovo progetto QGIS con un semplice layer vettoriale

Modificare un progetto QGIS¶

  1. Cliccare Desktop ‣ Desktop GIS ‣ Quantum GIS.

  2. Selezionare File ‣ Apri progetto... dalla barra del menu.

  3. Selezionare QGIS-NaturalEarth-Example.qgs e cliccare press Open.

    • Dovreste vedere una mappa del mondo.

  4. Attivare ne_10m_populated_places_simple nella legenda dei layer

    • Il layer “Populated places” viene visualizzato:

  5. Doppio click su ne_10m_rivers_lake_centerlines nella legenda.

  6. Cambiare il colore in un tono diverso di blu.

  7. Cliccare OK.

    • Ora i fiumi sono visualizzati con un nuovo colore.

Creare un nuovo progetto QGIS¶

  1. Selezionare File ‣ Nuovo progetto.
  2. Selezionare Layer ‣ Aggiungi vettore....
  3. Selezionare il file /home/user/data/natural_earth2/ne_10m_admin_0_countries.shp.
  4. Cliccare Open.
    • Dovreste vedere le nazioni del mondo.

Connettersi ad un database spaziale PostGIS¶

  1. Selezionare File ‣ Nuovo progetto.
  2. Selezionare Layer ‣ Aggiungi vettore PostGIS ....
    • Sono disponibili i database Natural Earth e OpenStreetMap.
  3. Selezionare OpenStreetMap e cliccare Connetti.
    • Apparirà una lista di tabelle del database.
  4. Selezionare planet_osm_roads e cliccare su Aggiungi.
    • Riferirsi a PostGIS Quickstart per informazioni più dettagliate.

Usare gli strumenti GRASS¶

  1. Selezionare File ‣ Nuovo progetto.
  2. Selezionare Plugins ‣ Gestione plugins....
  3. Scrivere grass nella casella Filtro e selezionare il plugin GRASS, quindi cliccare su Ok.
    • Appariranno una nuovo menu GRASS nel menu Plugins ed una nuova barra degli strumenti.
  4. Selezionare Plugins ‣ GRASS ‣ Apri mapset.
    • Il database GRASS viene automaticamente impostato sul percorso ~/grassdata.
  5. Selezionare the nc_basic_smp come Location e user1 come mapset, quindi cliccare su Ok.
  6. Selezionare Plugins ‣ GRASS ‣ Aggiungi raster GRASS.
    • Selezionare dal mapset PERMANENT la mappa elevation_shade e cliccare su Ok.
  7. Selezionare Plugins ‣ GRASS ‣ Aggiungi vettore GRASS.
    • Selezionare dal mapset PERMANENT la mappa elevation e cliccare su Ok.
    • Doppio click su elevation nella legenda ed impostare la trasparenza su 65%.
  8. Selezionare Plugins ‣ GRASS ‣ Aggiungi vettore GRASS.
    • Selezionare dal mapset PERMANENT la mappa roadsmajor e cliccare su Ok.

E’ possibile, inoltre, accedere ai potenti moduli GRASS per l’analisi spaziale.

  1. Selezionare Plugins ‣ GRASS ‣ Apri strumenti GRASS.
    • Apparirà una lunga lista di strumenti di analisi.
  2. Scrivere nviz nella casella Filtro e selezionare nviz - Visualizzatore 3D (NVIZ).
  3. Nella scheda del modulo NVIZ, selezionare elevation come nome del raster per l’elevazione e roadsmajor come vettore di overlay, quindi cliccare su Run.
  4. Si aprirà una nuova finestra NVIZ. Massimizzarla, impostare z-exag a 10.0, perspective a 15.0, height a 3500 e trascinare verso sinistra il punto di vista.
  5. Selezionare Visualize ‣ Raster surfaces ed impostare la risoluzione fine ad 1.

Usare la barra degli strumenti Sextante¶

  1. Selezionare Plugins ‣ Gestione plugins....
  2. Scrivere sextante nella casella Filtro e selezionare il plugin SEXTANTE. Cliccare su Ok.
    • Apparirà un nuovo menu ed una nuova barra deglistrumenti.
  3. Selezionare SEXTANTE ‣ SEXTANTE Toolbox.
    • Apparirà a destra una nuova barra ricca di strumenti di analisi.

Usare il plugin OpenStreetMap¶

  1. Aprire un terminale dal menu Lubuntu Applicazioni ‣ Accessori.
    • Fare una copia dei dati OSM nella cartella home:
cp data/osm/feature_city_CBD.osm.bz2 .
bzip2 -d feature_city_CBD.osm.bz2
  1. In QGIS, selezionare File ‣ Nuovo progetto.
  2. Selezionare Plugins ‣ Gestione plugins....
  3. Scrivere open nella casella Filtro e selezionare il plugin OpenStreetMap. Cliccare su Ok.
    • Apparirà un nuovo menu Web ed una nuova barra di strumenti.
  4. Selezionare Web ‣ OpenStreetMap ‣ Carica OSM da file.
  5. Selezionare il file feature_city_CBD.osm appena copiato nella cartella home.
  6. Attivare le caselle name, highway, e amenity.
  7. Selezionare il visualizzatore Medium scale e cliccare su Ok.
