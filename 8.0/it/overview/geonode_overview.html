GeoNode¶

Geospatial Content Management System¶

GeoNode è un Content Management System per dati geospaziali che permette la creazione, condivisione e l’uso collaborati di dati geospaziali. I set di dati possono essere caricati in diversi formati, le mappe possono essere modificate, cambiate di stile e aggregate attraverso strumenti utilizzabili dal browser, mappe e metadati possono essere pubblicati e ricercate, e possono essere ottenuti dagli utenti revisioni, voti e commenti.

GeoNode è costruito su: GeoServer, GeoExplorer, pycsw, Django, e GeoExt.

Caratteristiche principali¶

  • Ricerca dati spaziali
    • Potente motore di ricerca spaziale
    • Servizi OGC centralizzati
    • Catalogo dei metadati
  • Importazione e gestione dei dati geospaziali
    • Publicazione di dati raster, vector, e tavolari
    • Gestione dei metadati e documenti associati
    • Condivisione sicura o pubblica dei dati
    • Editor dei dati geospaziali con versioning
  • Mappe interattive
    • GeoExplorer GIS client
    • Editor grafico degli stili
    • Crea mappe interattive multi-layer
    • Condivide e integra mappe nelle pagine web
    • Stampa mappe come PDF
  • Collaborazione
    • Revisioni, voti e commenti sui dati
    • Gruppi di utenti
    • Flussi delle attività
    • Annunci e notifiche
  • ... e molto altro!

Standard implementati¶

  • Web Map Service (WMS)
  • Catalogue Service per il Web (CSW)
  • Web Feature Service (WFS)
  • Web Coverage Service (WCS)
  • Filter Encoding (FE)
  • Style Layer Descriptor (SLD)
  • Geography Markup Language (GML)
