TinyOWS 快速入门文档¶

　　TinyOWS 是一款高性能的事务性网络地理要素服务（WFS-T）系统。它使用 CGI 或 FastCGI 接口和 PostGIS 存储，轻量而易于部署。

　　本文演示：

  • 在 QGIS 渲染显示 WFS 图层
  • 在 QGIS 以 WFS-T 工具编辑 WFS 图层

以 QGIS 显示 WFS 图层¶

  1. 在桌面选择 Desktop ‣ Desktop GIS ‣ Quantum GIS 。
  2. 点击菜单 Layer ‣ Add PostGIS Layer... 。
    • 首先显示 PostGIS 原有图层。
  3. 选择 New 新建。
  4. 以 tinyows_demo 为连接名称。
  5. 点击 OK 。
  6. 点击 Connect 连接。
  7. 在表单列表中选择 france 。
  8. 点击 Add 添加。
  9. 放大至合适的比例。
  10. 选择 Plugins -> Manage Plugins -> WFS Plugin -> OK 。
    • 打开 WFS 图层插件
  11. 点击 Layer ‣ Add WFS Layer... 。
    • 现在从表单导入 WFS 图层。
  12. 点击 New 。
  13. 输入 tinyows 为名称（name），http://localhost/cgi-bin/tinyows 为 URL
  14. 点击 OK 。
  15. 点击 Connect 连接。
  16. 在图层列表选择 ows:france 。
  17. 选中 Only request features overlapping the current view extent （只获取当前显示范围内的要素）。
  18. 点击 OK 。
    • WFS 图层将显示：

使用 WFS-T 编辑数据¶

  1. 选择 Layer ‣ Toggle Editing 启动编辑。
  2. 选择 Edit ‣ Node Tool 节点工具。
  3. 将一个节点（十字线）拖动。
  4. 点击 Layer ‣ Save edits 保存。
  5. 取消勾选 tows:france 图层。
    • 新的数据将直接从 PostGIS 载入，以新的颜色显示。

其它信息¶

　　请访问 MapServer 主页的 TinyOWS 文档 了解进一步的信息。
