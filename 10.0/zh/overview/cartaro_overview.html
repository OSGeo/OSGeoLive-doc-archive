Cartaro¶

地理空间内容管理系统¶

　　Cartaro 为 Drupal Content Management System (CMS) 内容管理系统提供空间操作能力。通过简单的几次点击操作，Cartaro 用户就可以设置和运行 OGC 标准的网络服务，并生成相应的网页。Cartaro 使用 the PostGIS 数据库、 GeoServer 、GeoWebCache 缓存服务、 OpenLayers 网页客户端等可靠的开源技术，并同 Drupal CMS 系统结合。

　　Cartaro 为需要处理空间数据的 CMS 站点和需要轻量化空间数据基础服务（SDI）的组织设计，对用户配置和二次开发的要求很小。

Core Features¶

　　Cartaro 的 SDI 系统将 Drupal 同 PostGIS 和 GeoServer 紧密结合。主要特性有：

  • 几何空间数据类型的存储
  • 图形界面下的要素创建
  • 集成的在线数据编辑
  • 内嵌地图的数据发布
  • 地图布局与行为配置
  • 符号样式配置
  • 基于 OGC 标准网络服务（例如 WMS 和 WFS）的数据发布
  • 基于 GeoWebCache 的高性能服务
  • 对所有空间数据的权限管理
  • 内容发布与回退管理
  • 通过 GeoServer-GUI 获取元数据
  • 可由数千种 Drupal 模块或二次开发扩展

兼容标准¶

  • Web Map Service (WMS) 网络地图服务
  • Web Feature Service (WFS) 网络地理要素服务
  • Web Coverage Service (WCS) 网络地理场数据服务
  • Filter Encoding (FE) 过滤器编码
  • Style Layer Descriptor (SLD) 图层样式注记
  • Geography Markup Language (GML) 地理注记
