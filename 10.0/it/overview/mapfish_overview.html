MapFish¶

Web Mapping Framework¶

MapFish è un ambiente flessibile e completo per creare ricche applicazione webGIS. Si rileva alta produttività, e sviluppo di alta qualità.

MapFish è basato sull’ambiente Python Pylons . MapFish estende Pylons con funzionalità specificatamente geospaziali. Per esempio MapFish fornisce strumenti specifici per creare servizi web che permettono di interrogare ed editare oggetti geografici.

MapFish inoltre fornisce un completo toolbox JavaScript RIA-oriented, un ambiente JavaScript di test, e strumenti per comprimere il codice JavaScript. Il toolbox JavaScript è composto da ExtJS, OpenLayers , GeoExt JavaScript toolkits.

Standard implementati¶

MapFish soddisfa gli standards del Open Geospatial Consortium . Questo è possibile grazie a OpenLayers o GeoExt supportando diversi formati OGC, come WMS, WFS, WMC, KML, GML ecc.
