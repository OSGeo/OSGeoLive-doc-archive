TinyOWS Quickstart¶

TinyOWS ist ein hochperformanter, leichtgewichtiger und einfach einzurichtender Transaktionaler Web Feature Service (WFS-T),
der CGI und FastCGI Schnittstellen unterstützt

und PostGIS für die Datenspeicherung nutzt.

Dieser Quickstart Guide beschreibt folgende Schritte:

  • Einen WFS Layer in QGIS anzeigen
  • Einen WFS Layer mit dem WFS-T Protokoll in QGIS editieren

WFS Layer mit QGIS anzeigen¶

  1. Applications ‣ Desktop GIS ‣ Quantum GIS im Startmenü des Desktops wählen
  2. Wählen Sie Layer ‣ PostGIS Layer hinzufügen....
    • Wir zeigen zuersten den Original PostGIS Layer an
  3. Klicken Sie auf die Schaltfläche Neu.
  4. tinyows_demo als Name und Datenbank der Verbindung eingeben
  5. Klicken Sie auf die Schaltfläche OK.
  6. Klicken Sie auf die Schaltfläche Verbinden.
  7. Zeile france der Tabellen Liste auswählen
  8. Klicken Sie auf die Schaltfläche Hinzufügen.
  9. Zoomen Sie auf einen beliebigen Teilbereich hinein
  10. Wählen Sie Layer ‣ WFS Layer hinzufügen....
    • Jetzt fügen wir einen WFS Layer hinzu, der auf der selben Tabelle basiert
  11. Klicken Sie auf die Schaltfläche Neu.
  12. tinyows als Name und http://localhost/cgi-bin/tinyows als URL einfüllen
  13. Klicken Sie auf die Schaltfläche OK.
  14. Klicken Sie auf die Schaltfläche Verbinden.
  15. ows:france in der Layer-Liste auswählen
  16. Setzen Sie einen Haken bei der Option Nur Objekte, die den aktuellen Ausschnitt scheiden laden.
  17. Klicken Sie auf die Schaltfläche OK.
    • Der WFS Layer wird jetzt angezeigt:

Daten editieren mit WFS-T¶

  1. Wählen Sie Layer ‣ Bearbeitungsstatus umschalten.
  2. Wählen Sie Bearbeiten ‣ Objekte überarbeiten.
  3. Verschieben Sie einen Randpunkt (Kreuz) an eine andere Position
  4. Wählen Sie Layer ‣ Änderungen speichern.
  5. Deselektieren Sie tows:france in der Layer Liste, um den WFS-Layer auszublenden
    • Sie sollten den eben editierten Linienzug in einer anderen Farbe sehen, direkt von PostGIS geliefert.

Wie geht es weiter?¶

Um mehr über TinyOWS zu erfahren, dient die Seite TinyOWS Documentation auf der MapServer Homepage.
