gvSIG Quickstart¶

Projektbeschreibung¶

“gvSIG ist ein Geographisches Informationssystem (GIS), das heißt, eine Desktopanwendung zum Erfassen, Speichern, Bearbeiten, Analysieren und Veröffentlichen jeglicher Art räumlich referenzierter geographischer Informationen und zum Lösen komplexer Fragestellungen in Verwaltung und Planung. gvSIG ist für seine benutzerfreundliche Bedienoberfläche bekannt, die den Zugriff auf die meisten gängigen Datenformate, sowohl auf Vektor- als auch auf Rasterbasis, ermöglicht. Es bietet eine breite Auswahl an Werkzeugen für die Bearbeitung geographischer Informationen (Abfragewerkzeuge, Layouterstellung, Geoprocessing, Netzwerk, etc.), und ist damit das ideale Werkzeug für alle Anwender aus Fachgebieten in denen raumbezogene Daten eine Rolle spielen. gvSIG 2011

Dieses Dokument zeigt wie man...

  • gvSIG öffnet und eine Ansicht erstellt;
  • Die Projektion für die Ansicht festlegt ;
  • Raster- und Vektordaten zu einer Ansicht hinzufügt;
  • In einer Ansicht navigiert;
  • Die Anzeige der Ebenen in einem View ändert;
  • Städte auswählt die sich in einer bestimmten Region befinden und diese in eine neue Datei exportiert;
  • Eine Karte mit einem Gitternetz erstellt; und
  • Das aktuelle Projekt speichert oder gvSIG beendet.
  • die Installation eines Add-On’s (OpenStreetMap Service)
  • Laden eines WTMS Layers

Note

Sollte die Benutzetoberfläche nicht mehr reagieren, versuchen Sie Alt+f, um zum Beispiel das Datei-Menü zu öffnen. Die Oberfläche sollte nun wieder reagieren. Dieser Fehler tritt nur auf der OSGeo-Live distribution. Bei normalen gvSIG Installationen kommt es nicht zu diesem Problem

gvSIG öffnen und eine Ansicht erstellen¶

Wählen Sie gvSIG im entsprechenden Anwendungsmenü (Geospatial ‣ Desktop GIS ‣ gvSIG) oder klicken Sie auf das entsprechende Symbol. Die Anwendung benötigt etwa eine Minute um zu starten.

  1. In der Standardeinstellung startet gvSIG mit einem leeren Projekt sowie einer leeren ausgewählten Ansicht. In diesem Fall jedoch startet gvSIG mit einem vorkonfiguriertem Projekt. Klicken Sie auf New Project oder legen sie ein neuse Projekt über File ‣ New Project an. Sollte eine Nachfrage zur Speicherung des aktuellen Projektes kommen, können diese mit Discard Changes verworfen werden.
  2. Klicken Sie auf den Button New (Neu) um eine Ansicht zu erstellen.

Die Projektion für die Ansicht festlegen¶

Öffnen sie den Dialog Eigenschaften der Ansicht¶

  1. Wählen Sie die neu erstellte Ansicht. In der Standardeinstellung hat sie den Namen Ohne Titel-0 - 0 aber es kann auch Ohne Titel-1 - 1 sein falls sie zuvor bereits eine Ansicht erstellt und wieder gelöscht haben.
  2. Wählen Sie View ‣ Properties, um den Dialog Eigenschaften Ansicht zu öffnen

Definieren Sie das Raumbezugssystem¶

  1. Klicken Sie auf den Button ... rechts vom Eintrag Current projection (Aktuelle Projektion). Damit wird der Dialog New CRS (Neues Raumbezugssystem (CRS)) geöffnet.
  2. Verwenden Sie die Dropdownliste für Type, und wählen Sie EPSG.
  3. Geben Sie den EPSG code für das Koordinatensystem ein. In diesem Beispiel verwenden wir EPSG=4326, dies ist die WGS84 2D Geographic Projection. Klicken Sie dann auf den Button Search (Suchen). Das gesuchte CRS erscheint in der Tabelle unterhalb des Suchfeldes.
  4. Wählen Sie OK um zum Dialog Eigenschaften Ansicht zurückzukehren.

Note

Beachten Sie dass der EPSG Code nun 4326 lautet und dass die Karteneinheiten nun in Grad angegeben sind.

  1. Klicken Sie auf :guilabel`OK` um zum Projektverwalter zurückzukehren.

Öffnen Sie die Ansicht¶

  1. Wählen Sie die neu erstellte Ansicht aus.
  2. Klicken Sie auf den Buttton Open (Öffnen) um die Ansicht zu öffnen.
  3. Das Ansichtsfenster ist in drei Bereiche unterteilt.
  4. Der Abschnitt links oben enthält eine List von Vektor-oder Rasterebenen die in der Ansicht verwendet werden (eine Art Inhaltsverzeichnis)
  5. Der linke untere Abschnitt zeigt -für Vektordateien- die Position der dargestellen Ansicht innerhalb der Ausdehnung der gesamten Datei
  6. Der rechte Bereich ist der Hauptanzeigebereich in dem Raster- und Vektordaten dargestellt werden.
  7. Wählen Sie das Symbol ‘Maximieren’ um den ganzen Bildschirm für die Darstellung der Ansicht zu nutzen.

Bereiche der Ansicht kennenlernen¶

  1. Das Fenster besteht aus drei Bereiche
  2. Oben links ist die Liste der Vektor- und Rasterlayer, die in der Ansicht verwendet werden. Diese Liste wird auch als “Table-of-Content” bezeichnet
  3. Unten links ist der Übersichtsbereich zu finden, worin der Ausschnitt des Hauptfensters dargestellt wird
  4. Der rechte Bereich ist das Kartenfenster, wo die Vektor- und Rasterdaten gerendert werden

Die Ansicht - Raster- und Vektordaten hinzufügen¶

Öffnen Sie den Dialog “Layer hinzufügen”¶

  1. Klicken Sie auf Add layer (Layer hinzufügen) in der Hauptwerkzeugleiste um den Layer Hinzufügen Dialog zu öffnen.
  2. Wählen Sie Add (Hinzufügen) im Dialog Layer Hinzufügen. In der Standardeinstellung heißt der erste Reiter File (Datei), somit erscheint der Dialog Öffnen.
  3. Folgen Sie dem Verzeichnisbaum bis zum Verzeichnis /usr/local/share/data/natural_earth2/
  4. Wählen Sie die Dateien, wie in der folgenden Abbildung zu sehen.

Note

Es werden standardmäßig alle unterstützen Formate angezeigt. Es können Raster- und Vektordaten zusammen ausgewählt werden, in dem man die Taste CRTL gedrückt hält.

  1. Wählen Sie OK um zum vorhergehenden Dialog zu gelangen
  2. Wählen Sie im Layer Hinzufügen Dialog auch hier OK, um zur Hauptansicht zu kommen.

Eine Rasterdatei auswählen¶

  1. Wählen Sie gvSIG Raster Driver aus der Dropdownliste. in der Standardeinstellung ist der Treiber für Shapefiles selektiert, das Verzeichnis kann deshalb leer erscheinen.
  2. Wählen Sie HYP_50M_SR_W.tif.
  3. Klicken Sie auf [OK] um zum Dialog Layer hinzufügen zurückzukehren.

Vektordateien auswählen¶

  1. Der Dialog Layer hinzufügen enthält nun als Eintrag die Rasterdatei.
  2. Klicken Sie auf den Button [Hinzufügen] um einige Vektordateien hinzuzufügen. Der Dialog Öffnen erscheint.
  3. Navigieren Sie im Verzeichnisbaum eine Ebene nach oben um ins Verzeichnis /usr/local/share/data/natural_earth zu wechseln.
  4. Wählen Sie gvSIG shp driver aus der Dropdown-Liste.
  5. Wählen Sie die Shape-Dateien 10m_admin_1_states_provinces.shp, 10m_populated_places_simple.shp und 10m_rivers_lake_centerlines.shp aus der Liste. Halten Sie die Contol-Taste gedrückt um mehrere Dateien gleichzeitig auszuwählen.
  6. Klicken Sie auf den Button [OK] um zum Dialog Layer hinzufügen zurückzukehren.
  7. Klicken Sie auf den Button [OK] im Dialog Layer hinzufügen um zurück zur Ansicht zu gelangen.

Die Ansicht- grundlegende Navigation¶

In eine Region zoomen¶

Zurück in der Hauptansicht sehen sie dass die Vektordateien nun der Rasterdatei überlagert sind. Die Farben in diesem Screenshot können sich je nach Benutzereinstllungen von denen auf Ihrem Bildschirm unterscheiden.

  1. In der Standardeinstellung ist das Werkzeug Zoom (+) in der Haupt-Werkzeugleiste aktiviert.
  2. Verwenden Sie die Maus um ein Auswahlrechteck um Südost-Australien zu zeichnen. Klicken Sie dazu in die linke obere Ecke in der die Auswahl beginnen soll, halten Sie die linke Maustaste gedrückt und ziehen Sie so die rechte untere Ecke des Rechtecks bis zur entsprechenden Position, um den gewünschten Bereich auszuwählen. Lassen sie dann die linke Maustaste los.

Feineinstellung der Navigation¶

Die Ansicht ändert sich automatisch um die Inhalte innerhalb des Auswahl.

  1. Die Breite der Layerliste kann mit der Maus angepasst werden um auch lange Dateinamen anzuzeigen.
  2. Passen sie Darstellung der Ansicht mittels der hervorgehobenen Werkzeuge an, so dass Victoria im Zentrum der Ansicht dargestellt wird.

Die Ansicht - Das Erscheinungsbild von Layern ändern¶

Ein Polygon transparent darstellen¶

  1. Selektieren Sie den Layer 10m_admin_1_states_provinces.shp mit der linken Maustaste und klicken Sie noch einmal mit der rechten Maustaste darauf.
  2. Selektieren Sie das Untermenü Eigenschaften.
  3. Der Dialog Layereigenschaften erscheint.

Die Darstellungssymbole für einen Vekorlayer ändern¶

  1. Wählen Sie den Reiter Symbole im Dialog Layereigenschaften.
  2. Klicken Sie auf [Symbol wählen] um den Symbol-Selektor zu öffnen.
  3. Entfernen Sie den Haken von der Box Farbe der Füllung um die Polygone transparent darzustellen.
  4. Klicken Sie auf den Button [OK] um zum Dialog Layereigenschaften zurückzukehren.

Die Beschriftung eines Vektor-Layers ändern¶

  1. Wählen Sie den Reiter Beschriftung im Dialog Layereigenschaften.
  2. Ändern Sie die Texteigenschaften so wie im Beispiel dargestellt, klicken Sie dann auf den Button [Anwenden].

Das Ergebnis¶

Beachten Sie dass dies eine sehr einfache Ansicht ist, die nur jeweils eine Punkt-, Linien- und Polygon-Vektordatei zeigt die einer Rasterdatei überlagert sind. Genau so einfach wäre es, ein Luftbild oder ein Digitales Geländemodell als Hintergrund für Vektordaten zu verwenden, oder andere Vektordaten in unterschiedlichen Formaten anzuzeigen.

Eine Übung für Anwender - Ändern sie die Symbole und die Beschriftung der Punkt- und Linienlayer¶

Ändern Sie Symbole, Farbe and Beschriftung für Flüsse und Städte so dass sie aussehen wie im folgenden Screenshot. Gehen Sie dazu den bisher beschriebenen Schritten entsprechend vor.

Einfache Analyse - selektieren Sie Städte in einer Region¶

Die Region auswählen¶

  1. Selektieren sie den Layer 10m_admin_1_states_provinces.shp mittels der linken Maustaste.
  2. Drücken Sie den Button ‘Auswahl durch Klick’ in der Haupt-Werkzeugleiste.
  3. Klicken Sie in das Polygon welches den State of Victoria darstellt. Das Polygon wird gelb oder nimmt, je nach Anwendereinstellungen, eine andere Farbe an.

Die Städte innerhalb einer Region auswählen¶

  1. Selektieren Sie den Layer 10m_populated_places_simple.shp mittels der linken Maustaste.
  2. Wählen Sie Ansicht > Elemente Auswählen > Auswahl durch Layer um den Dialog Auswahl durch Layer zu öffnen.
  3. Ändern Sie die beiden Auswahlkriterien mittels der Dropdown-Felder auf der linken Seite des Dialogs Auswahl durch Layer. Klicken Sie auf den Button [Neuer Datensatz] um Städte innerhalb des gewählten Polygons zu selektieren.
  4. Wählen Sie den Button [Abbrechen] im Dialog Auswahl durch Layer um zur Ansicht zurück zu gelangen.

Heben sie die Auswahl für das Polygon auf um das Ergebnis zu sehen¶

  1. Wählen Sie den Layer 10m_admin_1_states_provinces.shp mit der linken Maustaste.
  2. Klicken Sie auf das Symbol ‘Auswahl aufheben’ in der Haupt-Werkzeugleiste.
  3. Sie sehen dass nur nur Städte innerhalb von Victoria ausgewählt sind.

Exportieren Sie die ausgewählten Städte in eine neue Shapedatei¶

  1. Selektieren Sie den Layer 10m_populated_places_simple.shp mittels der linken Maustaste.
  2. Wählen Sie Layer > Exportieren nach to... > SHP um mit dem Export zu beginnen.
  3. Der Dialog Exportieren nach... erscheint. gvSIG erkennt dass nur 26 Elemente selektiert sind und weist Sie darauf hin dass nur ein Teil der Shapedatei exportiert wird. Wählen Sie zum fortfahren den Button [Ja].
  4. Der Dialog Speichern öffnet sich. Geben Sie der Datei einem Namen und speichern Sie sie an einem Ort nach Wahl. Klicken Sie zum Fortfahren auf den Button [Speichern]. Beachten Sie dass eine Warnmeldung erscheint falls bereits eine gleichnamige Datei existiert.
  5. Der Dialog Layer hinzufügen erscheint. Wählen Sie den Button [Ja] um die neue Shapedatei zur Liste der Layer hinzuzufügen.

Nur Städte innerhalb einer Region anzeigen¶

  1. Zurück in der Ansicht entfernen Sie den Haken vor dem Layer 10m_populated_places_simple.shp in der Layerliste.
  2. Ändern Sie Symbole und Beschriftung des layers test.shp wie im Screenshot unten dargestellt.
  3. Die Ansicht enthält jetzt nur Städte innerhalb des State of Victoria.
  4. Verwenden Sie das Symbol [Fenster schließen] um zum Projektverwalter zurück zu gelangen, damit wir eine Karte aus dieser Ansicht erstellen können.

Wie man eine Karte erstellt¶

Erstellen Sie eine neue Karte und öffnen Sie diese¶

  1. Wählen sie den Dokumenttyp Karte im Projektverwalter.
  2. Klicken Sie den Button [Neu] um eine Karte zu erstellen.
  3. Selektieren Sie die neu erstellte Karte. In der Standardeinstellung heißt dieseOhne Titel - 0.
  4. Klicken Sie auf den Button [Öffnen].
  5. Eine leere Karte erscheint in einem eigenen Fenster, der Name lautet Karte: Ohne Titel - 0. Beachten Sie dass eine Reihe von Punkten gleichmäßig über das Kartenbild verteilt ist. Man nennt dies Gitter oder Hilfslinien und sie helfen beim Erstellen und Formatieren der Karte indem sie nahe gelegene Punkte einrasten können.
  6. Wählen Sie das Symbol [Fenster maximieren] um den ganzen Bildschirm zur Darstellung zu nutzen.

Fügen sie eine Ansicht mit einem Gitternetz ein¶

  1. Klicken Sie auf das Symbol “Ansicht einfügen” in der Haupt-Werkzeugleiste
  2. Erzeugen Sie ein Auswahlrechteck mit dem Umgriff der Karte auf der Seite, indem Sie auf die leere Karte klicken, die Maustaste gedrückt halten und ziehen, bis die gewünschte Größe des Kartenausschnittes erreicht ist. Dadurch wird der Dialog Eigenschaften Ansichtsrahmen geöffnet.
  3. Wählen Sie die zuvor erstellte Ansicht.
  4. Setzen Sie einen Haken bei der Option Gitternetz anzeigen dadurch wird ein Gitternetz erzeugt.
  5. Klicken Sie auf den Button [Konfigurieren] um den Dialog Eigenschaften Koordinatengitter zu öffnen.
  6. Im Dialog Eigenschaften Koordinatengitter ändern Sie das Intervall für das Gitter auf 1.0, das heißt ein Grad zwischen den Linien für Breitengrade und Längengrade.
  7. Ändern Sie das Format des Gitters (der Übersichtlichkeit halber ist es besser Linien statt Punkte zu wählen).
  8. Vergrößern sie die Schriftgröße auf 14.
  9. Klicken Sie auf den Button [OK] um zum Dialog Eigenschaften Ansichtsrahmen zurück zu gelangen, dann drücken Sie [OK] um den Dialog zu schließen und zur Karte zurückzukehren.

Was kann man noch zur Karte hinzufügen?¶

  1. Wählen Sie Karte > Eigenschaften aus dem Hauptmenü um die Dialogbox Eigenschaften Karte zu öffnen. Entfernen Sie den Haken von Gitternetz anzeigen und klicken Sie auf [OK]. Das Gitternetz sollte nun von der Karte entfernt sein und die Karte sollte so aussehen wie unten dargestellt.
  2. Zusätzliche Elemente wie eine Maßstabsleiste oder ein Richtungspfeil können mit Hilfe der Symbole in der Haupt-Werkzeugleiste oder mit den Untermenüs im Menü Karte hinzugefügt werden.
  3. Die Karte kann gedruckt oder zur weiteren Verwendung in anderen Dokumenten als PDF oder Postscript exportiert werden.
  4. Wählen Sie das Symbol [Fenster schließen] um zum Projektverwalter zurück zu gelangen

Speichern Ihres Projektes oder Schließen von gvSIG¶

  1. Projekte können zur späteren Bearbeitung mittels der Option Datei > Speichern unter... gespeichert werden, oder
  2. man kann Projekte schließen indem man die Option Datei > Schließen auswählt.
