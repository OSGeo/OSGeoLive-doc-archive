Guia ràpida de TinyOWS¶

TinyOWS és un Transactional Web Feature Service (WFS-T) d’alt rendiment, que alhora és lleuger i fàcil d’implementar, emprant una interfície CGI o FastCGI i usant PostGIS per a l’emmagatzematge de dades.

Aquesta guia ràpida descriu com:

  • mostrar una capa WFS en un projecte de QGIS
  • editar una capa WFS emprant WFS-T des de QGIS

Mostrar una capa WFS en un projecte de QGIS¶

  1. Clic Desktop ‣ Desktop GIS ‣ Quantum GIS.
  2. Clic Layer ‣ Add PostGIS Layer....
    • Primer mostrem la capa PostGIS original
  3. Prémer New.
  4. Posar tinyows_demo com a nom de la Connexió i de la base de dades
  5. Prémer OK.
  6. Prémer Connect.
  7. Escollir france a la llista de la taula
  8. Prémer Add.
  9. Zoom in
  10. Escollir Plugins -> Manage Plugins -> WFS Plugin -> OK
    • Habilita el plugin WFS Layer.
  11. Clic Layer ‣ Add WFS Layer....
    • Ara afegim una capa WFS basada en la mateixa taula
  12. Prémer New.
  13. Posar tinyows com a nom i http://localhost/cgi-bin/tinyows com URL
  14. Prémer OK.
  15. Prémer Connect.
  16. Escollilr ows:france a la llista de capes
  17. Marcar Only request features overlapping the current view extent.
  18. Prémer OK.
    • Ara es mostra la capa WFS:

Editar la capa amb WFS-T¶

  1. Clic Layer ‣ Toggle Editing.
  2. Clic Edit ‣ Node Tool.
  3. Mou un vértex (creu) a una posició diferent
  4. Clic Layer ‣ Save edits.
  5. Amaga la capa desmarcant tows:france a la llista de capes
    • Ara hauries de veure el vial editat en un color diferent, tret directament de PostGIS.

I ara què?¶

Per obtenir més informació sobre TinyOWS, un punt de partida és la User Documentation a la pàgina d’inici de TinyOWS.
