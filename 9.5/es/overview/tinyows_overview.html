TinyOWS¶

Web Feature Service¶

TinyOWS es un WFS transaccional (WFS-T) de alto rendimiento, ligero y sencillo de desplegar, mediante interfaz CGI o FastCGI y usando PostGIS para el almacenamiento de datos.

TinyOWS normalmente es usado junto con MapServer para proporcionar WFS-T y servicios WFS rápidos para clientes QGIS y/o OpenLayers. Es usado en producción en organizaciones de todo el mundo, incluyendo agencias gubernamentales. TinyOWS implementa los estándares WFS 1.0 y 1.1 de manera estricta, y ha pasado todas las pruebas unitarias OGC cite (~ 1000 pruebas unitarias).

Características principales¶

  • WFS transaccional (WFS-T)
  • Interfaz CGI y FastCGI
  • Conexión con PostGIS
  • Formato de salida en GML 2.1.2, 3.1.1 y GeoJson 1.0
  • Configuración mediante ficheros de configuración de Mapserver, permitiendo un único fichero para ambas aplicaciones.

Estándares implementados¶

  • WFS 1.0 y WFS 1.1: perfiles Básico y Transicional
  • FE 1.0.0 y FE 1.1.0
  • GML 2.1.2 y 3.1.1 Perfil Simple (SF-0)
