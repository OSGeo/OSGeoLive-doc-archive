在硬盘安装 OSGeo-Live¶

　　OSGeo Live 可以同其它的 Ubuntu 发行版一样从 DVD 或 ISO 镜像安装至硬盘。你也可以在虚拟机中执行安装。

系统要求¶

  • 20 GB 。

安装 OSGeo-Live¶

选择 Install RELEASE

密码为“user”

选择语言

选择时区

选择键盘布局

选择安装分区。请注意，该分区原有的数据会被 删 除 。

默认设置如下

  • Name = “user”
  • username = “user”
  • password = “user”
  • computer name = “osgeolive”
  • 自动登录

选择 Install/安装

等待安装完成即可。
