Установка OSGeo-Live на жёсткий диск¶

OSGeo-Live может быть установлен на жёсткий диск Вашего компьютера тем же способом, что и другие дистрибутивы, основанные на Ubuntu.

Этот процесс также может быть запущен внутри виртуальной машины с образа DVD или ISO для создания виртуальной машины с OSGeo-Live.

Системные требования¶

  • 20 GB свободного места на жёстком диске

Установка OSGeo-Live¶

Выберите Install RELEASE

Введите пароль “user”

Выберите язык

Выберите временную зону

Выберите клавиатуру

Укажите диск, на который хотите установить OSGeo-Live. Будьте осторожны, это может удалить существующую разметку диска.

Значения по умолчанию:

  • Имя = “user”
  • логин = “user”
  • пароль = “user”
  • компьютер = “osgeolive”
  • Входить автоматически

Выберите Install

Выпейте чашку кофе, пока приложение устанавливается.

Это всё.
