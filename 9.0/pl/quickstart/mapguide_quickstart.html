MapGuide Quick Start¶

This guide describes how to:

  • Getting MapGuide Up and Running
  • Stylization of spatial data using Maestro

Getting MapGuide Up and Running¶

  1. Manually install MapGuide

Note

Due to space constraints on the DVD, MapGuide is currently not installed.

  1. Open a browser to the address http://localhost:8008/mapguide/phpviewersample/ajaxtiledviewersample.php to view the sample application

MapGuide Maestro¶

  1. To start Maestro, go to Desktop->Server, click icon MapGuide Maestro, this will open Maestro login GUI, as shown in following screenshot. Login to http://localhost:8008/mapguide/mapagent/mapagent.fcgi with username “Administrator” and password “admin”
  1. Click Ok to go to Maestro main GUI, as shown in following screenshot.
  1. Expand tree view on left panel, go to Samples->Sheboygan->Layers, double click Buildings, the layer editor for Buildings will open in the right panel, as shown in the following screenshot:
  1. Scroll down the right panel to show the Layer Style. On the Featurestyle column, click the ... cell to bring up the Style Editor, as shown in the following screenshot:
  1. in Foreground Color dropdown list, select Green, click Ok to close Style Editor. In Maestro Main GUI, click the Disk icon to save.
  1. to preview the change, open a browser to address http://localhost:8008/mapguide/phpviewersample/ajaxtiledviewersample.php or click the Preview icon in the tool bar from Maestro main GUI), which will open a browser to the above address. In the browser map view, Zoom in to building level, the building color now is Green.
