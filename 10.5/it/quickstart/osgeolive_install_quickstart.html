Installare OSGeo-Live nel Hard Disk¶

OSGeo-Live può essere installata nel vostro hard disk, con lo stesso metodo con il quale una distribuzione Ubuntu è installata.

Questo processo può anche essere eseguito dentro una Macchina Virtuale per creare una Macchina Virtuale OSGeo-Live da un DVD o immagine ISO.

Requisiti di Sistema¶

  • 20 GB of spare Hard Disk Space

Installare OSGeo-Live¶

Selezionare Install RELEASE

Inserire password=”user”

Selezionare il languaggio

Selezionare il fuso orario

Selezionare la tastiera

Specificare su che disco si vuole installare. (Attenzione, questo può eliminare partizioni esistenti installate).

Le impostazioni di base sono:

  • Nome = “user”
  • Nome utente = “user”
  • password = “user”
  • Nome del computer = “osgeolive”
  • Accesso automatico

Selezionare Install

Avviare e andare a prendere un tazza di caffè mentre l’installazione è in avanzamento.

Questo è tutto.

Tip

Suggerimenti per l’installazione Nel caso state installando un nome utente diverso da “user”, l’applicazione tomcat richiede di avere i permessi per il comando come descritto in http://trac.osgeo.org/osgeo/ticket/1153
