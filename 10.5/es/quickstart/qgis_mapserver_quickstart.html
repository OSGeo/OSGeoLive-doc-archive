Guía de inicio rápido de QGIS Server¶

QGIS Server ofrece un servicio web de mapas (WMS) basado en la librerías de la aplicación de escritorio de Quantum GIS (QGIS).

Esta Guía de Inicio Rápido describe como:

  • actualizar un proyecto de QGIS y representarlo con QGIS Server
  • crear un nuevo proyecto de QGIS y abrir el WMS directamente en un navegador

Iniciar el visor GeoExt¶

OSGeo-Live contiene un ejemplo de visor web de mapas basado en el marco de desarrollo Javascript GeoExt y usando QGIS Server para representar el mapa.

  • Clic en Geospatial ‣ Web Services ‣ QGIS Server

Debe ver una ventana del navegador con un visor web de mapas mostrando un mapa del mundo.

Ahora se va a cambiar el estilo del mapa usando QGIS.

Editar el proyecto de QGIS¶

  1. Clic en Geospatial ‣ Desktop GIS ‣ Quantum GIS.
  2. Seleccione Archivo ‣ Abrir proyecto... en la barra de menús.
  3. Seleccione QGIS-NaturalEarth-Example.qgs y pulse Abrir.
    • Debe ver el mismo mapa del mundo representado por QGIS. Ahora se va a cambiar el estilo de los ríos y ver el resultado en el visor web de mapas.
  4. Doble clic en 10m_rivers_lake_centerlines en el árbol de capas.
  5. Cambie el color en Opciones de línea exterior a un diferente tono de azul.
  6. Pulse Aceptar.
    • Los ríos ahora son representados en un nuevo color.
  7. Guarde el proyecto desde Archivo ‣ Guardar proyecto.
  8. Volver abrir el navegador web
  9. Acercar
    • Ahora debe ver los ríos representados con el nuevo color.

Crear un nuevo proyecto de QGIS¶

  1. Volver abrir QGIS
  2. Seleccione Archivo ‣ Nuevo proyecto.
  3. Clic en Capa ‣ Añadir capa vectorial....
  4. Navegar a los datos /home/user/data/natural_earth/10m_admin_0_countries.shp.
  5. Pulse Abrir.
    • Debe ver todos los países del mundo.
  6. Guarde el proyecto desde Archivo ‣ Guardar proyecto.
  7. Use el nombre de archivo /home/user/world.qgs y pulse Guardar.
  8. Abra la URL WMS.
    • Debe ver los países de mundo, esta vez representados por QGIS Server.

¿Qué sigue ahora?¶

Para aprender más sobre QGIS Server, puede seguir los artículos del blog de Linfiniti o 3LIZ.
