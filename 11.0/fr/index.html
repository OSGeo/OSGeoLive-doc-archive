Bienvenue sur OSGeo-Live 11.0¶

OSGeo-Live est un DVD, une clé USB bootable ou une image disque virtuelle indépendante basée sur Lubuntu, qui vous permet d’essayer une large variété de logiciels opensource géospatiaux sans avoir à installer quoi que ce soit. Il repose entièrement sur des logiciels libres, ce qui permet de le redistribuer, dupliquer gratuitement et de le passer à n’importe qui.

Il fournit des applications pré-configurées pour un éventail de cas d’utilisations géospatiaux, incluant le stockage, la publication, le visionnage, l’analyse et la manipulation de données. Il contient aussi des jeux de données de tests et de la documentation.

Pour essayer ces applications simplement:

  1. Insérez le DVD ou la clé USB dans l’ordinateur ou la machine virtuelle.
  2. Redémarrez l’ordinateur. (vérifiez l’ordre de démarrage des périphériques si nécessaire)
  3. Appuyez sur la touche “Entrée” pour démarrer et vous connecter.
  4. Essayez les applications en les lançant à partir du menu “Geospatial”.

OSGeo-Live est un projet de la fondation OSGeo . La fondation OSGeo est une organisation à but non lucratif supportant le développement, la promotion et l’éducation aux logiciels Open Source géospatiaux.

Guides de démarrage rapide¶

  • Démarrer avec le DVD OSGeo-Live
  • Changer la langue ou le type de clavier
  • Installer OSGeo-Live sur votre disque dur
  • Lancer OSGeo-Live dans une machine virtuelle
  • Créer une clé USB bootable OSGeo-Live

Présentation¶

Une présentation d’une demi-heure, mettant en avant toutes les applications de OSGeo-Live, est disponible avec les slides, les scripts et un résumé.
