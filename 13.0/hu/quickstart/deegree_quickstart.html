deegree gyorstalpaló¶

degree egy térkép (és vektor adat) szolgáltató Java alkalmazás más kliensekben megjelenítéshez.

Ez a gyorstalpaló leírja:

  • deegree web szolgáltatások indítása
  • deegree web szolgáltatások konfigurálása a deegree szolgáltatások konzolon keresztül
  • deegree-inspire-workspace használata kiindulópontként

Projekt leírás¶

deegree egí nyílt forráskódú szoftver a térinformatikai adat infrastruktúrákhoz és a térinformatikai web-hez.

Elérhető komponensek:

  • Web szolgáltatások: WFS, WMS, WMTS, CSW, WPS

Ez a gyorstalpaló leírja:

  • Indítsa el a deegree 3 webservices a Live DVD-n
  • Használja a deegree 3 services console-t a példa konfigurációk aktiválásához
  • További információk

A deegre 3-ről¶

deegree 3 a deegree Java keretrendszer új generációja GIS alkalmazásokhoz és OGC szolgáltatások implementációjához. Web alapú téradat infrastruktúra (GDI) építésére használható.

deegree3 web szolgáltatások indítása¶

Válassza a «Start deegree»-t a Geospatial ‣ Web Services ‣ deegree menüből. Az alkalmazás néhány pillanat múlva elindul.

deegree3 web szolgáltatások konzol¶

Amikor először indítja a deegree-t a firefox is megnyílik (különben nyissa meg és a http://localhost:8033 címre irányítsa a böngészőt). Kérjük várjon, amíg a deegree 3 services console-t nem látja. Ez egy egyszerű web alapú GUI a deegree workspace-nek nevezett konfiguráció fájlok szerkesztéséhez.

deegree 3 munkaterületek¶

A deegree 3 olyan konfigurációkat használ, mely megfelel minden deegree 3 alapú web szolgáltatásnak. A konfigurációkat úgynevezett workspae-kben tároljuk.

A deegree services console-on belül egy fejezetet találhat a használatra kész workspace-k kezelésére. Kattintson a bal oldali «workspaces»-re a fejezetbe belépéshez.

Most a futó deegree példány workspace kezelőjében van. Láthatja melyik workspace aktív (csak egy lehet aktív egyidőben), inaktív lokálisan elérhető workspace-ket és hivatalos workspace-ket importálhat, ha van internet kapcsolata.

Kérjük most indítsa el a deegree-inspire-workspace-t.

deegree 3 wokspace-k az OSGeoLive GIS lemezen¶

  • deegree-workspace-inspire

Ez a workpace egy minta INSPIRE nézet és letöltés szolgáltatás konfigurációt tartalmaz. Ez egy tranzakciós WFS-ből (2.0.0 és 1.1.0) és egy WMS-ből (1.3.0 és 1.1.1) áll, melyek az az I. függelék összes adattémái közül három réteget szolgáltatnak. Tartalmaz néhány holland adminisztrációs egységet, telkeket és címeket. A WFS ebben a példában mint egy INSPIRE letöltés szolgáltatás (közvetlen hozzáférés) szerepel és az eredménye helyes, harmonizált INSPIRE GML. Továbbá többféle kérdés típust és lekérdezést támogat.

Javaslat

Ez a workspace összhngban lévő INSPIRE GML helyi fájlokat tölt be. Ez könnyen megváltoztatható PostGIS, Oracel Spatial vagy Microsoft SQL server háttérre.

A deegree-workspace-inspire aktiválása után egy egyszerű térkép klienst nyithat meg a «see layers»-re kattintva. A szolgáltatást az adatok és egy alaptérkép (OpenStreetMap) megjelenítésére konfigurálták.

A jobb oldali +*-ra kattintva megnyitja az összes elérhető réteget tartalmazó réteg fát. Itt kapcsolhatja be a deegree WMS által szolgáltatott INSPIRE réteget.

Javaslat

A térkép kliens az OpenLayers-re épül. A térkép eltolásához tartsa a egér bal gombját lenyomva és húzza a térképet. A nagyítás az egérgörgő vagy a bal oldali navigációs elemek segítségével lehetséges.

További információ és lépésről-lépésre oktatóanyag az INSPIRE munkaterületről megtalálható a deegree dokumentációban.
