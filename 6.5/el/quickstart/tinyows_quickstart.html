Εγχειρίδιο Γρήγορης Εκκίνησης TinyOWS¶

Το TinyOWS είναι ένα λογισμικό υψηλών επιδόσεων που υλοποιεί την διαδικτυακή υπηρεσία Transactional Web Feature Service (WFS-T) σε ένα ελαφρύ και εύκολα διαχειρίσημο περιβάλλον, με τη χρήση τεχνολογιών όπως CGI ή FastCGI και σε βάση δεδομένων PostGIS για αποθήκευση των χωρικών δεδομένων.

Αυτός ο οδηγός περιγράφει πως:

  • να παρουσιάσετε ένα επίπεδο WFS στο λογισμικό QGIS
  • να επεξεργαστείτε ένα επίπεδο WFS στο QGIS μεσω της τεχνολογίας WFS-T

Παρουσίαση δεδομένων επιπέδου WFS στο λογισμικό QGIS¶

  1. Κάντε κλίκ στο μενού Desktop ‣ Desktop GIS ‣ Quantum GIS.
  2. Κάντε κλίκ στο Layer ‣ Add PostGIS Layer....
    • Εδώ εμφανίζεται το αρχικό επίπεδο της PostGIS
  3. Επιλέξτε New.
  4. Πληκτρολογήστε pgrouting σαν όνομα σύνδεσης (Connection name) και Database
  5. Πιέστε OK.
  6. Πιέστε Connect.
  7. Επιλέξτε ways στην λίστα πινάκων
  8. Επιλέξτε Add.
  9. Κάντε μεγέθυνση
  10. Επιλέξτε από το μενού Plugins -> Manage Plugins -> WFS Plugin -> OK
    • Ενεργοποιούμε το πρόσθετο για επίπεδα WFS.
  11. Κάντε κλίκ στο μενού Layer ‣ Add WFS Layer....
    • Τώρα προσθέτουμε ένα επίπεδο WFS βασισμένο στον ίδιο πίνακα
  12. Επιλέξτε New.
  13. Συμπληρώστε tinyows σαν όνομα και http://localhost/cgi-bin/tinyows σαν URL
  14. Επιλέξτε OK.
  15. Πιέστε Connect.
  16. Επιλέξτε ows:ways στη λίστα επιπέδων
  17. Ενεργοποιήστε Only request features overlapping the current view extent.
  18. Πιέστε OK.

Επεξεργασία δεδομένων μέσω WFS-T¶

  1. Πιέστε Layer ‣ Toggle Editing.
  2. Πιέστε Edit ‣ Node Tool.
  3. Μετακινήστε εναν κόμβο σε άλλη θέση
  4. Πιέστε Layer ‣ Save edits.
  5. Κρύψτε το επίπεδο απενεργοποιώντας tows:ways στη λίστα επιπέδων
    • Μπορείτε να δείτε τις επεξεργασίες με άλλο χρώμα μέσω της απευθείας σύνδεσης με την PostGIS.

Τι ακολουθεί;¶

Για να μάθετε περισσότερα σχετικά με το TinyOWS, ένα καλό σημείο έναρξης είναι το εγχειρίδιο χρήσης στην κεντρική ιστοσελίδα TinyOWS.
