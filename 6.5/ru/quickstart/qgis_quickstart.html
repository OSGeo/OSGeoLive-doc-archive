QGIS: начало работы¶

Quantum GIS (QGIS) — это дружественная к пользователю настольная ГИС с открытым исходным кодом, позволяющая отображать, редактировать и анализировать данные, управлять ими, а также создавать печатные карты.

Этот документ описывает, как:

  • обновить существующий проект QGIS;
  • создать новый проект QGIS с простым векторным слоем.

Редактирование проекта QGIS¶

  1. Нажмите Desktop ‣ Desktop GIS ‣ Quantum GIS.

  2. Выберите в меню File ‣ Open Project....

  3. Укажите файл QGIS-NaturalEarth-Example.qgs и нажмите Open.

    • Вы должны увидеть карту мира.

  4. В списке слоёв установите галку напротив 10m_populated_places_simple

    • Отобразятся города:

  5. Дважды щёлкните на 10m_rivers_lake_centerlines в списке слоёв.

  6. Измените цвет в Outline Options на другой оттенок голубого.

  7. Нажмите OK.

    • Реки должны отобразиться другим цветом.

Создание нового проекта QGIS¶

  1. Выберите File ‣ New Project.
  2. Нажмите Layer ‣ Add Vector Layer....
  3. Перейдите к набору данных /home/user/data/natural_earth/10m_admin_0_countries.shp.
  4. Нажмите Open.
    • Вы должны увидеть границы стран мира.

Подключение к пространственной базе данных PostGIS¶

  1. Выберите File ‣ New Project.
  2. Выберите Layer ‣ Add PostGIS Layers....
    • Доступны две базы данных Natural Earth и OpenStreetMap.
  3. Выберите базу данных OpenStreetMap и нажмите Connect.
    • Появится список таблиц базы данных.
  4. Выберите planet_osm_roads и нажмите Add.
    • Более подробное упражнение можно найти в PostGIS Quickstart.

Использование инструментов GRASS¶

  1. Выберите File ‣ New Project.
  2. Выберите Plugins ‣ Manage Plugins.
  3. Введите grass в окно фильтра и выделите модуль GRASS. Затем нажмите Ok.
    • Появится вложенное меню GRASS в меню Plugin, а также новая панель инструментов.
  4. Выберите Plugins ‣ GRASS ‣ Open mapset.
    • Путь к базе данных GRASS будет автоматически установлен в ~/grassdata.
  5. Выберите область nc_basic_smp и набор данных user1, затем нажмите Ok.
  6. Выберите Plugins ‣ GRASS ‣ Add GRASS raster layer.
    • Из набора данных PERMANENT выберите карту elevation_shade и нажмите Ok.
  7. Выберите Plugins ‣ GRASS ‣ Add GRASS raster layer.
    • Из набора данных PERMANENT выберите карту elevation и нажмите Ok.
    • Дважды щёлкните на слое elevation в списке слоёв QGIS и установите его прозрачность в 65%.
  8. Выберите Plugins ‣ GRASS ‣ Add GRASS vector layer.
    • Из набора данных PERMANENT выберите карту roadsmajor и нажмите Ok.

Вы также можете использовать множество аналитических модулей GRASS.

  1. Выберите Plugins ‣ GRASS ‣ Open GRASS tools.
    • Появится длинный список инструментов анализа.
  2. Введите nviz в поле фильтра и выберите NVIZ. Возможно, вам нужно будет изменить размер окна, потянув его за угол.
  3. В появившейся вкладке модуля выберите карту elevation в качестве источника данных о высоте. Затем отключите кнопку с изображением прямогольника с красными углами справа от названия карты, чтобы использовать границы и разрешение карты. Правильно заданная рабочая область — ключевое действие в обработке растров в GRASS.
  4. Выберите векторный слой roadsmajor и нажмите Run.
  5. После того как откроется окно NVIZ, максимизируйте его и при помощи ползунка в окне компаса установите точку обзора.
  6. Выберите Visualize ‣ Raster surfaces и установите разрешение fine в 1, затем нажмите зеленую кнопку DRAW в верхней левой части окна и дождитесь окончания отрисовки.

Использование инструментов Sextante¶

  1. Выберите Plugins ‣ Manage Plugins.
  2. Введите sextante в поле фильтра и выберите модуль SEXTANTE. Нажмите Ok.
    • Появится новый пункт меню Analysis.
  3. Выберите Analysis ‣ Sextante Toolbox.
    • В правой части окна появится новая панель с различными инструментами анализа.

Использование модуля OpenStreetMap¶

  1. Запустите эмулятор терминала из меню Xubutu Applications ‣ Accessories.
    • Скопируйте данные OSM в домашний каталог:
cp data/osm/feature_city_CBD.osm.bz2 .
bzip2 -d feature_city_CBD.osm.bz2
  1. В QGIS, выберите File ‣ New Project.
  2. Зайдите в Plugins ‣ Manage Plugins.
  3. Введите open в поле фильтра и выберите модуль OpenStreetMap. Нажмите Ok.
    • В меню Web появится новое подменю, также появится новая панель инструментов.
  4. Зайдите в Web ‣ OpenStreetMap ‣ Load OSM from file.
  5. Укажите файл feature_city_CBD.osm, который вы только что скопировали в домашний каталог.
  6. Активируйте переключатели name, highway и amenity.
  7. Выберите тип отрисовки Medium scale и нажмите Ok.
