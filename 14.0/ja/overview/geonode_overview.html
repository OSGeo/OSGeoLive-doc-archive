GeoNode¶

地理空間コンテンツ管理システム¶

GeoNode は地理空間データのためのコンテンツ管理システムであり、地理空間データの作成、共有、および共同使用の仕組みを提供します。データセットはさまざまな形式でアップロードでき、マップはブラウザベースのツールを使用して編集、スタイル設定、集計を行うことができ、マップとメタデータを公開して検索することができ、レビュー、評価、コメントをユーザから取得することができます。

GeoNodeは、GeoServer、GeoExplorer、pycsw、Django、およびGeoExtに基づいて構築されています。

主な機能¶

  • 空間データ探索
    • 強力な空間検索エンジン
    • 統合OGCサービス
    • メタデータカタログ
  • 地理空間データのインポートと管理
    • ラスタ、ベクタ、テーブルデータをパブリッシュする
    • メタデータと関連ドキュメントを管理する
    • データを安全に共有する、あるいは公開する
    • バージョン管理された空間データエディタ
  • インタラクティブマッピング
    • GeoExplorer GISクライアント
    • スタイル編集画面
    • マルチレイヤインタラクティブマップを作成する
    • マップを共有してWebページに埋め込む
    • マップをPDFとして印刷する
  • コラボレーション
    • データのレビュー、評価、コメント
    • ユーザのグループ
    • アクティビティストリーム
    • アナウンスと通知
  • などなど ...

実装されている標準規格¶

  • Web Map Service (WMS)
  • Catalogue Service
  • Web Feature Service (WFS)
  • Web Coverage Service (WCS)
  • Filter Encoding (FE)
  • Styled Layer Descriptor (SLD)
  • Geography Markup Language (GML)
