OTB¶

イメージ処理¶

ORFEO Toolbox Library (OTB ) は、高性能イメージ処理ライブラリーです。主にレーダー、人工衛星、航空写真などの大きなリモートセンシング画像を処理するのに使われます。OTB は、光学およびレーダーのイメージ (三次元の側面、セグメンテーション、分類、変更検出、テクスチャー分析、パターン・マッチング、および光学/レーダーの補完性 ) を提供するツールを提供しており、これらの多くは性能が限定的なノートPCでも実行できます。

OTB には同梱アプリケーションとともに配布されています。それは、リモートセンシング用の拡張可能な従来型のツールと 、複数のレイヤを可視化させたりメニュー項目から OTB の処理ツールを呼び出せる高速なビューアであるMonteverdiという完全に統合されたアプリケーションです。

OTB は、ORFEO/Pleiades付属プログラムの メソドロジカル・パートの枠組みにもとづき、フランス宇宙機関 (CNES) の資金提供をうけて、 2006年から活発に開発されている。これは、 ITK イメージ処理ライブラリーに基づいており、オープンソースとして配布されます。

主な機能¶

  • 画像アクセス: ほとんどのリモートセンシングラスターフォーマット ( JPEG2000 形式を含む)、メタデータアクセス、および視覚化のための最適化された読み取り/書き込みアクセス
  • DEM、地図の再投影、およびセンサーモデルの改良によるオルソ化
  • 光学式および SAR 調整
  • 画像融合
  • フィルタリング: ぼかし、ノイズ除去、強調...
  • フィーチャー抽出: POI、整列、線分...
  • オブジェクト抽出
  • 大規模画像セグメンテーション: 領域拡張、平均シフト、水位、レベルセット...
  • トレーニング用のサンプル選択フレームワーク
  • 種別: OpenCV 、 Shark、 libSVM 、またはカスタム・インプリメンテーション ( K - 平均、マルコフ・ランダム・フィールド、 SVM、ランダム・フォレスト、 ...) からくる、教師ありアルゴリズムと教師なしアルゴリズムの両方。
  • OpenCV (SVM、ランダム・フォレスト、 ...) から機械学習メソッドを使用した回帰
  • オブジェクト・ベースのイメージ分析
  • 地理空間分析
  • 光学画像からの立体構造の復元
  • SAR データ分析 : 較正、偏光分析
  • 変更の検出。
