Instalar OSGeoLive en el Disco Duro¶

OSGeoLive puede ser instalado sobre su Disco Duro de la misma manera que una distribución de Ubuntu es instalada.

Este proceso también se puede ejecutar dentro de una Máquina Virtual para crear una Máquina Virtual permanente de OSGeoLive a partir de un de DVD o una imagen ISO.

Requerimientos del Sistema¶

  • 20 GB de espacio en Disco Duro de Repuesto

Instalar OSGeoLive¶

Selecione Install RELEASE

Introduzca la contraseña=»user»

Seleccione el lenguaje

Seleccione la zona horaria

Seleccione el teclado

Especifique el disco en el cual desea instalar. (Cuidado, esto puede eliminar una partición existente).

Valores predeterminados para ajustes son:

  • Nombre = «user»
  • nombre de usuario= «user»
  • contraseña = «user»
  • nombre de la computadora = «osgeolive»
  • Iniciar sesión automáticamente

Selecionar Install

Vaya por una taza de café mientras la instalación está en progreso.

Eso es todo.

Truco

Consejos de instalación En caso de que esté instalando utilizando un nombre de usuario distinto de «usuario», las aplicaciones tomcat requieren tener permiso para el comando service como se describe en https://trac.osgeo.org/osgeo/ticket/1153
