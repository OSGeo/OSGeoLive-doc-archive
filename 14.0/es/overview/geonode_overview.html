GeoNode¶

Sistema de Gestión de Contenido Geoespacial¶

GeoNode es un Sistema de Gestión de Contenido para datos geoespaciales que permite la creación, intercambio y uso colaborativo de datos geoespaciales. Los conjuntos de datos pueden cargarse en varios formatos, los mapas pueden ser editados, diseñados y agregados a través herramientas basadas en navegador, mapas y metadatos pueden ser publicados y buscados, las revisiones, calificaciones y comentarios pueden ser capturados de los usuarios.

GeoNode se construye sobre: GeoServer, GeoExplorer, pycsw, Django y GeoExt.

Características Principales¶

  • Descubrimiento de datos espaciales
    • Potente motor buscador espacial
    • Servicios OGC federados
    • Catálogo de metadatos
  • Importar y gestionar datos geoespaciales
    • Publicar datos tabulares, raster y vector
    • Gestión de metadatos y documentos asociados
    • Compartir datos con seguridad o públicamente
    • Editor de datos geoespaciales con versiones
  • Cartografía interactiva
    • Cliente GeoExplorer SIG
    • Editor de estilo gráfico
    • Crea mapas interactivos de múltiples capas
    • Compartir e integrar mapas en páginas web
    • Imprimir mapas en formato PDF
  • Colaboración
    • Revisión, evaluación y comentario en los datos
    • Grupos de usuarios
    • Corrientes de la actividad
    • Anuncios y notificaciones
  • … y más!

Estándares implementados¶

  • Web Map Service (WMS)
  • Catalogue Service
  • Web Feature Service (WFS)
  • Web Coverage Service (WCS)
  • Filter Encoding (FE)
  • Styled Layer Descriptor (SLD)
  • Geography Markup Language (GML)
