GeoExt¶

Un kit de herramientas de JavaScript para aplicaciones de mapeo web enriquecidas¶

GeoExt es de código abierto y permite la creación de aplicaciones SIG de escritorio a través de la web. Es un marco JavaScript que combina la funcionalidad SIG de OpenLayers con la interfaz de usuario de la librería ExtJS proporcionada por Sencha.

Los componentes de la interfaz de usuario incluyen un panel de mapa, un panel de árbol de capas, una cuadrícula de características y un mapa general. Los componentes GeoExt se pueden usar como cualquier otro componente ExtJS y se pueden configurar y ampliar fácilmente. Las clases de nivel inferior también están disponibles, lo que permite que las características y capas de OpenLayers se utilicen como modelos de datos ExtJS y que se serialicen varios tipos de capas de mapa.

GeoExt3 tiene doble licencia. La licencia que elija depende de la licencia del mar o de trabajo ExtJS que use con él. Si está utilizando la versión GPL v3 del marco ExtJS, la usará en GPLv3. Si posee una licencia comercial Sencha ExtJS, tiene derecho a usar GeoExt3 bajo la licencia BSD.

Características Principales¶

  • Componentes de la IU
    • Panel de mapas
    • Mapa general
    • Ventanas emergentes
    • Renderizador de funciones
    • Árbol de capas, leyendas y orden de capas de arrastrar y soltar
  • Modelos de Datos y Tiendas
    • Integración del modelo de datos OpenLayers con el modelo de datos ExtJS que permite la clasificación y el filtrado de características utilizando componentes ExtJS
    • Tiendas y modelos de ExtJS para características y capas de OpenLayers
  • Varios
    • Integración con Impresión de MapFish y el Módulo de Impresión GeoServer
    • Serializadores de capa (WMS, WMTS, XYZ)
