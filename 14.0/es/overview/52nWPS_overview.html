52 North WPS¶

Acerca de¶

El Web Processing Service (WPS) 52°North permite el acceso enables web web a algoritmos de procesamiento geoespacial proporcionados por Sextante, ArcGIS Server, R, GRASS 7 o funciones personalizadas desarrolladas. Los algoritmos pueden ser tan simples como determinar la diferencia en casos de influencia entre dos temporadas diferentes, o tan complicados como un modelo de cambio climático global.

Características Principales¶

Principales funciones de GIS:

  • SimpleBufferAlgorithm o Algoritmo Simple de Búfer
  • DouglasPeuckerAlgorithm o Algoritmo de Douglas Peucker
  • IntersectionAlgorithm o Algoritmo de Intersección
  • AddRasterValuesAlgorithm o Agregar Algoritmo de Valores de Ráster
  • CoordinateTransformAlgorithm o Algoritmo de Transformación de Coordenadas

algoritmos Backend disponibles a través de:

  • Sextante
  • Servidor ArcGIS
  • R
  • GRASS 7

Formatos de entrada y de salida compatibles:

  • Datos vectoriales: GML2, GML3, archivos SHP, WKT (Texto bien conocido), DGN, KML, JSON
  • Datos raster: Geotiff AsciiGrid, NetCDF, JPEG, PNG, HDF-EOS, ERDAS HFA
  • Outputs o Salidas pueden ser almacenadas en GeoServer/MapServer

Un cliente basado en navegador

Estándares implementados¶

  • Web Processing Service (WPS)
  • GML2 SimpleFeatures de OGC
  • GML3 SimpleFeatures de OGC
