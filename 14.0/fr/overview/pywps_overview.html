PyWPS¶

Service de géotraitement via le web¶

PyWPS fournit un accès web a des opérations spatiales personnalisées (aussi appelés Processes) via le standard Web Processing Service (WPS) .

Les process sont écrits dans le langage de programmation Python et peuvent intégrer des outils tels que GRASS GIS, R, GDAL/OGR, proj et d’autres bibliothèques avec des liens Python. Les données sources peuvent être fournies avec des requêtes WPS ou accédées à partir de données sur le serveur WPS.

Fonctionnalités principales¶

  • petit et léger
  • Facile et rapide à installer et configurer
  • Implémenté en Python pur
  • s’intègre sans heurt avec l’écosystème Python de paquets:
    • GRASS 7
    • R
    • GDAL/OGR
    • Proj
    • Numpy
    • Shapely

Standards implémentés¶

Standards OGC :

  • WPS 1.0.0
