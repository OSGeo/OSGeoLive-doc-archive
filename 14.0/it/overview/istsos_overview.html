istSOS¶

Strumento di gestione dei dati¶

istSOS è uno strumento di gestione dei dati di sensori che permette la raccolta, gestione e pubblicazione di osservazione di monitoraggio utilizzando lo standard Sensor Observation Service (SOS) dell’Open Geospatial Consortium (OGC).

istSOS implementa rigorosamente lo standard SOS 1.0.0 ed ha superato i test OGC CITE (Compliance + Interopability Testing + Evaluation).

Caratteristiche principali¶

  • Pubblica dati di sensori in accordo con lo standard Sensor Observation Service (SOS).
  • Amministra sensori e dati attraverso una interfaccia web intuitiva.
  • Utilizza una completa API RestFul per accedere alle funzionalità di IstSOS da client esterni per creare grafici interattivi, visualizzare un sensore su una mappa dinamica oppure scrivere qualche script di manutenzione.
  • Ricevi notifiche via mail, twitter o altro social media quando i dati di un sensore raggiungono specifiche condizioni.
  • Autenticazione ed autorizzazione dell’utente con differenti livelli di accesso (amministratore, gestore della rete, gestore dei dati e ospite)-
  • Crea Procedure Virtuali che sembrano normali stazioni ma i cui dati derivano da elaborazioni al volo di altri sensori.
  • Associa indici di qualità ad ogni osservazione grazie a test di validazione incorporati.
  • Modifica online dei dati con un’interfaccia semplice e un calcolatore avanzato di osservazioni.

Standard implementati¶
