GeoNode¶

Content Management System Geospaziale¶

GeoNode è un Content Management System per dati geospaziali che permette la creazione, condivisione e l’uso collaborativo di dati geospaziali. I set di dati possono essere caricati in diversi formati, le mappe possono essere modificate, cambiate di stile e aggregate attraverso strumenti utilizzabili dal browser, mappe e metadati possono essere pubblicati e ricercate, e gli utenti possono caricare revisioni, voti e commenti.

Geonode è creto utilizzando: GeoServer, GeoExplorer, pycsw, Django e GeoExt.

Caratteristiche principali¶

  • Ricerca dati spaziali
    • Potente motore di ricerca spaziale
    • Servizi OGC centralizzati
    • Catalogo dei metadati
  • Importazione e gestione dei dati geospaziali
    • Publicazione di dati raster, vettoriali e tabellari
    • Gestione dei metadati e dei documenti associati
    • Condivisione sicura o pubblica dei dati
    • Editor di dati geospaziali con versioning
  • Mappe interattive
    • Client GeoExplorer GIS
    • Editor grafico degli stili
    • Crea mappe interattive multi-layer
    • Condivide e integra mappe nelle pagine web
    • Stampa mappe come PDF
  • Collaborazione
    • Revisioni, voti e commenti sui dati
    • Gruppi di utenti
    • Flussi delle attività
    • Annunci e notifiche
  • … e molto altro

Standards implementati¶

  • Web Map Service (WMS)
  • Catalogue Service
  • Web Feature Service (WFS)
  • Web Coverage Service (WCS)
  • Filter Encoding (FE)
  • Styled Layer Descriptor (SLD)
  • Geography Markup Language (GML)
