actinia¶

Processamento geospaziale in High Performance Computing¶

Actinia è una REST API a codice aperto per il processamento scalabile, distribuito ad alte performance di da geografici che usa GRASS GIS per i compiti computazionali.

Fornisce un REST API per processare immagini satellitari, serie temporali di immagini satellitari, qualsiasi raster con relazione geografica e dati vettoriali.

Caratteristiche principali¶

  • API REST per processare immagini satellitari, serie temporali di immagini satellitari, dati raster e vettoriali.
  • Visualizza il turorial.
